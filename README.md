# chatbot_project
Chatbot coded in Python.

Full Setup:

1. Project Structure:
chatbot_project/
├── chatbot/
│   ├── __init__.py
│   ├── chatbot.py  (original CLI version)
│   └── gui.py      (new GUI version)
├──venv/

2. Activate & Install:
cd chatbot_project
.\venv\Scripts\Activate.ps1  # Activate venv
pip install nltk             # Install in venv

3. Run
python -m chatbot.chatbot
